# PawMate

A polished, responsive prototype for an AI-powered dog care and community product.

## Local development

```bash
npm install
npm run dev
```

Then open [http://localhost:3000](http://localhost:3000).

## Routes

- `/` — landing page
- `/login` — account creation prototype
- `/pawpal` — interactive AI care dashboard
- `/pawmatch` — searchable shelter dog directory
- `/pawmatch/[dogId]` — dog profiles
- `/shelters/sacramento-animal-shelter` — shelter profile
- `/community` — filterable local community board
- `/community/[postId]` — community post details

All content is local mock data; no backend or environment variables are required.
